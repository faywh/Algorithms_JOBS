## 2 阅读工具

大部分资料是Markdown格式的，为了有最好的阅读体验，建议下载typora-markdown阅读器：

https://typora.io/

由于部分文档中有大量公式推导，需要使用内联公式，请在安装好阅读器后进行如下设置：

```
文件->偏好设置

- Markdown扩展语法
  - 内联公式
  - 下标
  - 上标
  - 高亮
  - 图表

- 数学公式
  - 自动添加序号

以上选项都打勾
```
